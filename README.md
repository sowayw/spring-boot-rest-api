# spring-boot-rest-api (Learning project) 

This is a personal pet project created to practice Java, Spring Boot, and PostgreSQL.

***

## Tech Stack

* **Java 21**
* **Spring boot 4.0.6(Web, JPA)**
* **PostgreSQL**

## Getting Started 
***

1. Clone the repository:
```shell
git clone https://github.com/sowayw/spring-boot-rest-api.git

cd spring-boot-rest-api
```
2. Create PostgreSQL database and update credential in 'application.properties'.


3. Run the application using:

#### Linux/MacOS
```bash
./mvnw spring-boot:run
```

#### Windows
```shell
.\mvnw spring-boot:run
```

4. Open Swagger UI to test endpoints: [http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html)
